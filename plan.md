# Jar of Love Premium A+ 页面策划

## 已确认策略

- Hero: `A PRINT FOR EVERY MOMENT`
- 核心逻辑:多印花 × 多场景 × 多搭配
- 黑色承担双大图锚点,酒红作为暖色点缀
- 四款核心印花每款至少出镜 2 次,且使用不同场景或景别
- 每屏至少出现两种花色信息,不得形成单花色孤屏
- 模块顺序沿用仓库固定 01–07 结构

## 模块策划与唯一素材分配

| Module | 单一任务 | 主要素材 | 辅助花色 | 密度 |
|---|---|---|---|---|
| 01 Hero | 3 秒内建立“露肩上衣 + 多印花”认知 | `color-black-front-02.jpg` | indigobark / greyfloral | Low |
| 02 Details | 解释交叉肩带与领口结构 | 黑色领口、黑色肩带、charcoalblush 肩带 | charcoalblush | Medium |
| 03 Fabric/Fit | 证明垂坠与高低摆 | 黑色垂坠、黑色侧面、黑色背面 | 酒红侧面 | Medium |
| 04 Scenes | 建立六花色专属场景矩阵 | 6 张场景图 | 全矩阵 | High, strict grid |
| 05 Styling | 展示不同花色对应不同搭配 | 3 张差异化 Flat-lay | greyfloral / black / indigobark | Medium |
| 06 Size | 提供量法示意与尺码决策入口 | 黑色七分身 | 四印花小色卡 | High, structured |
| 07 Print Gallery | 四印花双景别收官 | 8 张实拍/Approved 印花图 | 四核心印花 | High, gallery |

## 客户疑虑对应

1. 这款和普通短袖有什么不同? → 露肩交叉带 Hero + 细节。
2. 袖部开口结构会不会看不懂? → Module 02 结构特写。
3. 衣长与垂坠是否修饰臀胯? → Module 03 正侧背证明。
4. 不同印花适合什么场合? → Module 04 场景矩阵。
5. 多花色是否只能配同一套单品? → Module 05 三组差异搭配。
6. 如何判断尺码与量法? → Module 06 量法示意;具体尺寸表待后台数据。
7. 四款印花有什么区别? → Module 07 双景别画廊。

## 页面制作限制

- 不使用 `Draft` 素材。
- 不新增生成素材。
- Module 06 不编造具体尺寸数值,仅展示 BUST / LENGTH / SLEEVE 量法并提示核对后台尺码表。
- 不使用价格、评论引用、绝对化词或竞品识别元素。
