# STYLE.md — ANRABESS 上衣类 A+ 设计规范(基于 18 张参考图提炼)

> 用途:所有 modules/*.html 必须遵循本规范。学习的是布局/配色逻辑/排版节奏,
> 禁止复制 ANRABESS 的 Logo、文案原句、模特照片。

## 1. 配色 Tokens
```css
--color-bg-light: #F4F2EE;    /* 主背景:暖米白/浅燕麦,几乎每个模块的底色 */
--color-bg-card: #FFFFFF;     /* 白色卡片区块,叠在米白底上做层次 */
--color-bg-warm: #EAE5DE;     /* 辅助底:浅驼灰,用于分隔或角标区 */
--color-text-primary: #1A1A1A;/* 标题:近黑 */
--color-text-body: #4A4A4A;   /* 正文深灰 */
--color-accent: #8B6F55;      /* 点缀:驼棕,用于手写体、下划线、小标签 */
```
- 整体为低饱和大地色系:米白、燕麦、驼色、棕、酒红、藏青、军绿
- 页面明暗节奏:浅底为主,靠**服装颜色和街拍环境**制造深浅变化,不用深色大底块
- 高饱和色只出现在产品本身(玫红背心、酒红上衣),背景永远克制

## 2. 字体层级(1倍尺寸 1464px 宽下)
| 用途 | 风格 | 规格 |
|---|---|---|
| 主标题 | 衬线体(Playfair Display / EB Garamond 类),全大写,字距+2~4px | 56–72px, 400–600 |
| 副标题/模块名 | 衬线体全大写 (如 SIZE CHART, DETAILED DESCRIPTION) | 36–44px |
| 手写点缀词 | Script 手写体(Fashion / Casual / Basic Tops) | 40–60px, 驼棕或深灰 |
| 正文说明 | 无衬线 (Helvetica/Arial 类) | 22–28px, #4A4A4A |
| 细节标注 | 无衬线小字 + 圆点符号引导 (● V Neck) | 24–28px |
- 衬线大标题 + 手写体点缀 + 无衬线小正文 的三层混排是核心识别特征

## 3. 构图与版式规则
- **拼贴式构图(核心)**:每屏 2–4 张模特图错落叠放,常用手法:
  - 圆形/胶囊形裁切窗(细节特写、场景切换)
  - 白边"贴纸/宝丽来"相框,轻微旋转 2–5°
  - 图片之间允许压叠 10–20%
- **图文比例**:图片占 75–85%,文字量极少,一屏只讲 1 个信息
- **产品细节三宫格**:三张等宽特写(领口/袖口/面料)横排 + 圆点标签,几乎每页必有
- **面料特写**:布料拧成漩涡状的微距图 + 成分文字(如 93% Rayon 7% Elastane)
- **搭配平铺(Flat-lay)**:配饰单品(包、耳环、墨镜、鞋、牛仔裤)白底抠图散排,组成 Collocation/Occasions Styling 区
- **颜色变体条**:圆角色块横排 + COLOR SELECTION 标签,或彩色布料叠放实拍图
- **留白**:模块边缘安全边距 ≥ 60px,标题与图之间 ≥ 40px

## 4. 页面叙事结构(模块顺序公式,18 张图高度一致)
1. **品牌横幅**:顶部一行小字 slogan(自有品牌请用自己的 slogan)
2. **Hero 主图**:品类大标题(衬线全大写)+ 2–3 张模特街拍拼贴 + 一句卖点
3. **细节三宫格**:领型 / 袖型 / 织法特写 + 圆点标注
4. **面料模块**:漩涡布料微距 + 成分 + 特性图标(Breathable/Soft/Stretchy)
5. **场景模块**:SUITABLE SCENARIOS / OCCASIONS,4 张不同场景街拍横排(通勤/逛街/度假/居家)
6. **搭配模块**:Flat-lay 配饰平铺,传达"一衣多穿"
7. **颜色矩阵**:色块条或多色实拍
8. **尺码表(必备,权重极高)**:SIZE CHART(IN) 标题 + Product Measurements + Body Measurements 双表 + STRETCH/FIT TYPE 游标标尺 + 一张标注 Bust/Length/Sleeve 量法的模特图 + 洗涤说明一行
9. **品牌矩阵收尾**:From the brand 交叉引流轮播

## 5. 模特与摄影基调(供 photo-brief 使用)
- 模特:20–35 岁欧美女性,棕/金长波浪发,妆容自然,姿态松弛(插兜、撩发、回眸、行走抓拍)
- 下装固定用**牛仔裤/牛仔短裤**做中性搭配,让上衣成为唯一主角
- 场景:欧式街道、白墙建筑、咖啡店、居家沙发、海边度假;自然光,暖调,轻微过曝的通透感
- 每页至少 1 组"正面+侧面+背面"三视角
- 构图预留:Hero 图需一侧留 35–40% 低复杂度背景供叠标题

## 6. 文案基调
- 短语化、关键词堆叠,不写长句(Design - Henley/Short Sleeve/Crewneck/Slim Fit)
- 卖点词库:Soft, Stretchy, Lightweight, Breathable, Slim Fit, Ribbed Knit, Versatile
- 每个模块 1 个标题 + ≤2 行说明;信息密度靠图不靠字
- 尺码表数据必须与后台属性完全一致

## 7. 数值化自检清单(每张成品图对照)
- [ ] 背景是否为 #F4F2EE 系暖米白(禁止纯白大底 #FFFFFF 直接做整屏底)
- [ ] 主标题是否衬线全大写、是否 ≥56px
- [ ] 是否有手写体点缀词(每屏最多 1 处)
- [ ] 图片占比是否 ≥75%,文字是否 ≤3 行
- [ ] 拼贴是否有圆形裁切或白边相框元素
- [ ] 每个非表格模块至少出现 3 个照片视窗/裁切区域,其中至少 1 个允许出血到画布边缘或被相邻照片压叠
- [ ] 连续两个非表格模块不得重复"左文右图"模板;必须切换为全出血、居中拼贴、四栏场景、斜向贴纸墙等不同构图
- [ ] 手写体/标签/水印文字必须参与画面装饰:每屏至少 1 处大号脚本字或半透明品牌字母压在照片/背景上,不能全部停留在独立左栏
- [ ] 模特或产品主体必须撑起画面:Hero/场景/搭配类模块中模特高度应接近画面高度的 70–95%,避免大面积空墙和空气感
- [ ] 可使用虚线箭头、#OOTD/#DETAILS 标签、Ins 风边框、小圆点/星形点缀等轻装饰,但不得遮挡领口、纽扣、袖长和面料纹理
- [ ] 跨模块通版设计:模块 01-03 必须先作为一张 1464×1800 连续画布整体设计,再精确切成 3 张 1464×600;模块 04-05 必须先作为一张 1464×1200 连续画布整体设计,再精确切成 2 张 1464×600;切片线要避开人脸、标题、纽扣细节,关键元素距切片线至少 60px
- [ ] 主模特一律使用抠图贴纸呈现:透明底人物沿人形轮廓加 8–12px 白色描边和柔和投影,不得把主模特放进矩形白边卡片
- [ ] 每张通版至少 1 个抠图模特的肢体或身体跨越 600px 模块切片线;跨线区域不得是人脸、标题、纽扣或关键文字
- [ ] 矩形白边卡片仅用于小图、细节图、照片便签或材质局部,不能替代主模特
- [ ] 配饰平铺组必须使用新生成并抠图的牛仔裤/手袋/耳环/鞋/墨镜等单品,散落排布替代部分文字胶囊
- [ ] 背景不得是单纯纯色底;使用柔和照片虚化底或带渐变的纹理底,相邻分区用大色块柔和过渡并保持米白体系
- [ ] 手写体必须作为标题主角:每张通版至少一处两行式大号棕色手写体标题,并配一处手绘卷曲虚线箭头连接视觉元素
- [ ] `assets/color-*.jpg` 仅作为服装还原基准,不得直接裁切用于最终画面;新生成素材统一入库 `assets/gen/`,命名 `gen-{用途}-{颜色}-{序号}.png`,并在 `assets/gen/INDEX.md` 记录用途和参考来源
- [ ] 尺码表是否含双表+游标标尺+量法示意图
- [ ] 尺码表量法示意必须使用真实模特抠图,用白色虚线直接标注 BUST/LENGTH/SLEEVE;表格行底色使用浅粉/浅驼交替条纹
- [ ] 边距是否 ≥60px
- [ ] 严禁用 CSS/SVG 绘制产品插画代替产品照片
- [ ] 所需照片在 assets/ 中不存在时,用灰色占位框标注"待素材",并在交付说明中列出缺失清单

## 8. 补充规律(基于第二批 15 张样本,总样本 33 张)

### 新增视觉元素
- **拱形裁切窗(Arch)**:半圆拱门形图框是仅次于圆形的高频裁切形状,常放中间主图
- **米驼色关键词横幅**:一条 #C9B9A3 系色带横贯全宽,上面均匀分布 3–4 个白色全大写关键词(如 FALL / FASHION / TRENDY / OUTFITS),用作模块之间的分隔线,几乎每页出现 1–2 次
- **笔刷底白色标签**:细节标注词(V NECK / SLEEVES / SOFT FUZZY)放在不规则白色笔刷涂抹底上,轻微旋转
- **场景图底部标签条**:4 张场景街拍每张底部压一条半透明色条,写场景词(CASUAL / SHOPPING / DAILY WEAR / OFFICE),全大写衬线体
- **颜色圆点色板**:除方形色块外也常用圆点排;点数与变体数一致

### 尺码表进阶细节
- 部分品类在 STRETCH / FIT TYPE 之外增加第三游标 **THICKNESS**(Lightweight–Middle–Thick),针织类必备
- 高频出现 **Model Information 框**:模特身高/胸围/腰围/臀围 + 所穿尺码,增强参考感
- 洗涤说明固定跟在尺码表正下方,一行小字

### 季节化微调(同一公式换皮肤)
- 秋冬款:场景换为秋叶街道、暖棕色调加重,可加落叶点缀元素;夏款:白墙/海边/高曝光
- 无论季节,底色体系和版式结构完全不变——季节感只靠**照片内容**传达,不改设计系统

### 竖版对比结论
33 张样本中模块顺序偏差极小,可确认为固定生产模板。制作自有产品时应严格按第 4 节顺序执行,不要自创结构。
